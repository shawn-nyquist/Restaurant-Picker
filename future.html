Introduction

The intent of this project was to create a simple application using pure JavaScript, HTML, and CSS and then expand into the new and more refined technologies. The idea is to mimic what could be considered "older" code and practice bringing it up to date. Some of the different frameworks to be used in the future include: VueJS, Angular, and React. I will also be trying to implement the google map API to do things like: choose options within a certain distance, show address of options, and show reviews of each option.
